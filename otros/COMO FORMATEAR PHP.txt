1.- Installar PHP
2.- https://code.visualstudio.com/docs/languages/php
    2.1.- To change the PHP settings, open your User or Workspace Settings (Ctrl+,) and type 'php'
     to filter the list of available settings.
    2.2.- To set the PHP executable path, select the Edit in settings.json link under PHP > Validate:
     Executable Path, which will open your user settings.json file. Add the php.validate.executablePath setting 
     with the path to your PHP installation:
    2.3.-   Windows:
            {
                "php.validate.executablePath": "c:/php/php.exe"
            }
            Ubuntu:
            {
                "php.validate.executablePath": "/usr/bin/php"
            }

3.- Instalar complemento de visual studio "PHP Intelephense"
